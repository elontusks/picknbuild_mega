# 18. Leaderboard, Contest, Badge, and Build Credit System

This system must be explicit.

## 18.1 Purpose

The system drives:

- Invitations
- Growth
- Engagement
- Retention
- Long-term participation

## 18.2 Invite Contest

- Users receive points when invited users actually sign up.
- Points are tied to real signups, not invite sends.

## 18.3 Leaderboards

Leaderboards must be offered at the following timescales:

- Daily
- Weekly
- All-time

| Leaderboard | Window |
| --- | --- |
| Daily | Resets every day |
| Weekly | Resets every week |
| All-time | Cumulative |

## 18.4 Rewards

Users earn:

- Build credits
- Badges
- Contest progress

Build credits are an internal reward currency that can apply toward a future build.

- An internal cap can exist.
- The public-facing system does not need to expose internal cap mechanics.

## 18.5 Grand Prize

- Top participants can win a free car as part of a contest system.

## 18.6 Badge System

Badges are awarded for:

- Invites
- Engagement
- Deals posted
- Community activity
- Platform participation milestones

| Badge category | Awarded for |
| --- | --- |
| Invites | Successful referrals |
| Engagement | Likes, comments, replies, active use |
| Deals posted | Sharing and curating deal content |
| Community activity | Helping peers, answering questions |
| Milestones | Platform participation thresholds |

## 18.7 Rule

> Contest system must be tied to real verified actions, not vanity interactions.

## 29.11.2 Referral and Contest Dashboard

Users must be able to view:

- Personal invite link or code
- Number of successful invites
- Total points earned
- Current leaderboard rank
- Earned build credits
- Badges
- Progress toward next reward
- Contest rules and deadlines

| Dashboard element | Purpose |
| --- | --- |
| Invite link/code | Share to invite others |
| Successful invites | Count of verified signups from this user |
| Total points earned | Cumulative points |
| Leaderboard rank | Current position |
| Earned build credits | Reward currency balance |
| Badges | Earned recognition items |
| Progress toward next reward | Visual/quantified progress |
| Contest rules and deadlines | Terms and timing |

---

## 29.11.8 Contest and Incentive Explainer

Users must be able to understand:

- How invites are counted
- How points are earned
- How build credits work
- How leaderboard ranking works
- How the free car prize is awarded

The explainer must cover each of these so the contest mechanics are transparent to every participant.
