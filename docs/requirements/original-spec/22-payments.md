# Payments Infrastructure

Stripe must be integrated as the payments backbone of the platform. All money flows must be tracked and auditable.

## Stripe Integration

Stripe must support the following money flows:

- picknbuild refundable deposit
- Dealer lead unlocks
- Dealer subscriptions
- Listing payments
- Auction service fees

| Flow | Type |
| --- | --- |
| picknbuild refundable deposit | One-time, refundable |
| Dealer lead unlocks | One-time, per-lead |
| Dealer subscriptions | Recurring |
| Listing payments | One-time, per-listing |
| Auction service fees | One-time, service |

## System Capabilities

The system must support:

- Payment processing
- Refunds
- Recurring subscriptions
- Failed payment handling
- Payment logs

## Rule

> All money flows must be tracked and auditable.
