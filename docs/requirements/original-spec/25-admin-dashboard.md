# Admin / Internal Dashboard

The admin / internal dashboard surfaces platform-wide operational state for the PicknBuild team.

## Required Views

The dashboard must show:

- Users
- Active leads
- Messaging activity
- Listings
- Feed activity
- Active deals
- Likely transaction outcomes
- Payment activity
- Dealer subscriptions
- Inventory ingestion health

## Summary

| Area | Description |
| --- | --- |
| Users | Platform user base and account state |
| Active leads | Currently active dealer leads |
| Messaging activity | On-platform message volume and flows |
| Listings | Live listings across supply sources |
| Feed activity | Engagement with the feed surface |
| Active deals | In-progress transactions |
| Likely transaction outcomes | Inferred transaction outcomes from §21 |
| Payment activity | Stripe-driven money flows |
| Dealer subscriptions | Active and lapsed dealer subscriptions |
| Inventory ingestion health | Status of scraping and ingestion pipelines |
