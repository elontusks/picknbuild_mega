# Location System

Location is a core variable, not a filter detail.

## Inputs

- ZIP code or city
- Auto-detect with manual override

## Outputs

- Distance to listing
- Shipping estimate
- Feasibility
- Timeline impact
- Local vs non-local prioritization

## What Location Must Affect

Location must affect:

- Dealer proximity
- Auction shipping
- PicknBuild delivery
- Private seller trust and convenience
- Price realism

## Rule

There is no valid pricing or timeline without location context.
