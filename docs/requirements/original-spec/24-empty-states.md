# Empty State Handling

If no suitable result exists, the platform must not dead-end. The user must always have a next step.

## Requirements

If no suitable result exists, the platform must instead:

- Show estimated paths
- Suggest alternatives
- Prompt "Found a Better Deal? Paste it here."

## Behavior Summary

| Condition | Required Behavior |
| --- | --- |
| No suitable result | Show estimated paths |
| No suitable result | Suggest alternatives |
| No suitable result | Prompt "Found a Better Deal? Paste it here." |

## Rule

> The user must always have a next step.
