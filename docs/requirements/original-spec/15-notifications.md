# 15. Notification System

## 15.1 Purpose

Notifications keep users, dealers, sellers, and PicknBuild informed and engaged.

## 15.2 Notification Categories

### User activity

- Likes
- Comments
- Replies

### Match alerts

- New match
- Cheaper option
- Better deal
- Lower monthly option
- Closer option

### Garage alerts

- Saved car changed
- Better alternative exists
- Path availability changed

### Feed alerts

- Relevant content from similar users
- Posts in user cluster
- Trending relevant content

### Dealer / seller alerts

- Lead created
- Listing interest
- Save activity
- Message activity

### Invite / leaderboard alerts

- Invite signup
- Rank changed
- Reward threshold reached
- Contest updates

### Category summary

| Category | Example triggers |
| --- | --- |
| User activity | Likes, comments, replies |
| Match alerts | New match, cheaper option, better deal, lower monthly option, closer option |
| Garage alerts | Saved car changed, better alternative exists, path availability changed |
| Feed alerts | Relevant content from similar users, posts in user cluster, trending relevant content |
| Dealer/seller alerts | Lead created, listing interest, save activity, message activity |
| Invite/leaderboard alerts | Invite signup, rank changed, reward threshold reached, contest updates |

## 15.3 Delivery Channels

- In-app
- Email
- Future-ready for push

## 15.4 Personalization

Notification delivery and selection must be personalized by:

- Location
- Saved cars
- User behavior
- Financial profile
- Decision stage

## 15.5 Frequency Control

- Batch low-priority alerts.
- Prioritize high-intent events.
- Avoid spam.

## Governing Rule

> Notifications must create action, not noise.

## 29.11.5 Notification Preferences

Users must have control over notification settings. The system must allow users to manage:

- Email vs in-app notifications
- Deal alerts
- Feed/social alerts
- Leaderboard and invite notifications
- Buyer/seller interaction alerts

| Preference | Controls |
| --- | --- |
| Channel | Email vs in-app |
| Deal alerts | On/off |
| Feed/social alerts | On/off |
| Leaderboard and invite | On/off |
| Buyer/seller interaction | On/off |
