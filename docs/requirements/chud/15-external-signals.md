# Dealer / Seller / External Signal Flows

## Dealer

When user interacts with a dealer listing:
- create lead signal
- support dealer re-entry / response through platform workflow

## Private Seller

When user expresses interest:
- seller gets a message/signal
- seller is prompted to join or continue in the platform
- this pulls external supply into internal flow

## External Listing Conversion Loop

1. user finds external deal
2. user compares it in PicknBuild
3. seller/dealer gets signal
4. seller/dealer returns into platform
5. system internalizes supply over time
