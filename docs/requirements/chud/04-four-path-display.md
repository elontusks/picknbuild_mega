# Four-Path Structure, Messaging & Display Requirements

## Path Headlines and Support Text

### Dealer
- **Headline / label:** Fastest Access
- **Support text:** Drive away quickly with dealer financing.

### Auction / DIY
- **Headline / label:** Lowest Price
- **Support text:** Insurance and salvage auctions. Handle bidding and repairs yourself.

### picknbuild
- **Headline / label:** No Credit Required
- **Support text:** No searching, no repairs, no credit or interest.

#### Branding rule
- display as `picknbuild`
- all lowercase
- only the `n` is red
- not the `K`

### Private Seller
- **Headline / label:** Lowest Cost
- **Support text:** Buy directly from the seller. Inspect and negotiate yourself.

Keep these simple and literal.

## Path Comparison Detail Requirements

At minimum, path cards must correctly represent:
- path label
- pricing style
- core trade-off
- sponsor board
- title badge
- relevant term/payment/risk logic
- timeline
